### üß™ Dev Usage

- Pipe interface: `/tmp/camrecorder.pipe`
- Save last 300 seconds:
  ```sh
  echo save:300 >> /tmp/camrecorder.pipe
  ```
- Stop the server:
  ```sh
  echo stop >> /tmp/camrecorder.pipe
  ```
- Press `Ctrl+C` to kill the server cleanly.
